# Битроид — Создание сайта под ключ с SEO-продвижением

Лендинг веб-студии **Битроид** — разработка сайтов с SEO-оптимизацией, аналитикой и поддержкой.

## Стек

- HTML5, CSS3, JavaScript
- [Swiper.js](https://swiperjs.com/) — слайдер кейсов
- jQuery — интерактивность (меню, FAQ-аккордеон)
- Google Fonts (Manrope)

## Структура

```
├── index.html          # Главная страница
├── css/
│   ├── main.css        # Основные стили + адаптив
│   └── normalize.css   # CSS Reset
├── js/
│   └── main.js         # Логика: меню, FAQ, слайдер
└── images/             # Изображения и favicon
```

## Секции

- **Hero** — заголовок, CTA-кнопки, 3D glass-figure
- **Аудитория** — карточки целевых сегментов
- **Что входит в услугу** — стеки разработки и SEO
- **Кейсы и проекты** — слайдер с фоновыми эллипсами
- **FAQ** — аккордеон с частыми вопросами
- **Контакт** — форма обратной связи
- **Футер** — навигация, контакты, соцсети

## Адаптивность

Полностью адаптивный дизайн: desktop → tablet (1180px, 992px) → mobile (640px, 480px, 320px).

## Запуск

Открыть `index.html` в браузере или запустить локальный сервер:

```bash
npx serve .
```
